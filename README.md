### Описание проекта

Проект представляет собой консольное казино с тремя азартными играми: Рулетка, Слот-машина и Блэкджек. Программа разработана на Python с использованием объектно-ориентированного подхода и модульной архитектуры.

**Что реализовано:**
1. **Рулетка** - классическая европейская рулетка с 37 секторами (0-36)
2. **Слот-машина** - игра с 3 барабанами и различными комбинациями символов
3. **Блэкджек** (21 очко) - упрощенная версия карточной игры против дилера

**Особенности проекта:**
- Система баланса игрока с начальным депозитом
- Валидация ставок и входных данных
- Цветной вывод в консоли для лучшей визуализации
- Статистика игровой сессии
- Возможность пополнения баланса

**Структура файлов:**
- `project.ipynb` - основной файл с кодом проекта
- `README.md` - документация проекта (этот файл)
- Видеопрезентация - демонстрация работы проекта

**Технические детали:**
- Использованы 4 обязательные функции: `main()`, `play_roulette()`, `play_slots()`, `play_blackjack()`
- Дополнительные классы для игр и управления балансом
- Обработка исключений и ошибок пользовательского ввода
- Комментарии ко всем функциям и основным блокам кода

Проект демонстрирует навыки работы с функциями, классами, обработкой пользовательского ввода и организацией кода в соответствии с принципами чистого программирования.
